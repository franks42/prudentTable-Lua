prudentTable.aGet (t, b, n)
Returns n element values as a list, starting at index b of table t. Range must not extend array boundaries. May return default values if so registered for t.

Parameters

  • t: a table instance
  • b: start of range (>=1)
  • n: number of element-values to return - b+n-1 <= len(t) - n default to 1 - n may be 0

Return value:

prudentTable.unpack (t, b, e)
Unpacks a table-array (from a previously packed-list) into a new list of values. Note that unpack relies on proper predentTable.setLen() registration, either thru prudentTable.pack() or some other prudentTable-function. It does NOT use any "t.n" that may be used to indicates the array-length. May return default values if so registered for t.

Parameters

  • t: table instance
  • b: start of range to copy - default to 1
  • e: end of range to copy - defaults to len(t)

Return value:

list of values representing t[b:e] - may include nils
