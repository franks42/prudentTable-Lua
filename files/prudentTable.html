pT.setLen (t, n, resetValues)
Set the length/size of the array component of table t and registers t as a prudentTable. Thru pT.setLen(), a Lua-table t is "registered" as a prudentTable. If size is increased, then existing map entries become part of the array. If size is decreased, then some array components become map entries. If resetValues==true then all new array/map elements get reset to nil. This reset is to avoid any inconsistencies caused by unexpected new map/array elements.

Parameters

  • t: An standard lua-table or prudentTable
  • n: Integer >= 0
  • resetValues: boolean (default true) - resets new array or map elements to nil.

Usage

  • local t = pT.setLen({a="a", b="b", 1,2,3,nil}, 4, false)
  • t = pT.setLen(t,5) ; pT.set(t,5,"five")

Return value:

prudentTable t (possibly modified - depending on n and resetValues)
